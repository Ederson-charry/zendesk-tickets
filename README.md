# Zendesk Tickets Dashboard - SmartQuick

Dashboard interactivo de KPIs de Zendesk para SmartQuick S.A.S.

## Archivos
- `zendesk_dashboard.html` - Dashboard estatico (5 tabs: Resumen, Mesa de Servicio, Niveles, Empresas, SLA)
- `zendesk_kpis.js` - Datos agregados (window.KPI_DATA), regenerado cada 4h por sync automatico
- `zendesk_kpis.json` - Mismos datos en JSON para consumo programatico
